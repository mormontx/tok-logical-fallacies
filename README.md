## üìö What's Included

This resource provides three interconnected components for teaching logical fallacies:

### 1. Student Texts
- **12 extended texts** across diverse, relatable topics
- Topics include: social media policy, school uniforms, climate change, homework debates, nutrition, athletics, technology in education, dress codes, standardized testing, school lunches, study methods, and extracurricular activities
- Each text contains **multiple embedded logical fallacies** for rich analysis
- Designed to engage students with realistic arguments they might encounter

### 2. Teacher's Annotated Guide
- **Password-protected** for educator-only access (Password: `TOK2024`)
- Fallacies highlighted in red with underlining
- Detailed annotations explaining why each represents flawed reasoning
- Comprehensive summaries with "Better Approach" suggestions
- Currently features detailed analysis of the first 3 texts as examples

### 3. Fallacy Reference Guide
- Complete explanations of **13 logical fallacy types**
- For each fallacy:
  - Clear definition
  - "How to Spot It" identification tips
  - Real-world examples
  - Why it's fallacious
  - Critical thinking questions to ask
- Color-coded cards for easy navigation

## üéØ Fallacy Types Covered

- **Ad Hominem** - Attacking the person instead of their argument
- **Appeal to Inappropriate Authority** - Citing experts outside their field
- **Appeal to Tradition** - "We've always done it this way"
- **Appeal to Popularity** - Bandwagon reasoning
- **Anecdotal Evidence** - Using personal stories as proof
- **False Cause** - Confusing correlation with causation
- **False Dilemma** - Presenting only two extreme options
- **Slippery Slope** - Claiming inevitable chain reactions
- **Hasty Generalization** - Jumping to conclusions from limited data
- **Cherry Picking** - Selecting only favorable evidence
- **Appeal to Uncertainty** - Dismissing science because it's not 100% certain
- **False Analogy** - Comparing incomparable things
- **Genetic Fallacy** - Judging ideas by their origin

## üöÄ How to Use

### For Teachers:
1. **Assign readings** from the student texts
2. Have students **identify fallacies** individually or in groups
3. Use the **reference guide** as a teaching tool or student resource
4. **Review answers** using the password-protected teacher guide
5. **Facilitate discussion** about why reasoning matters

### For Students:
1. Read each text carefully
2. Identify the main argument
3. Spot logical fallacies using the reference guide
4. Explain why the reasoning is flawed
5. Suggest how to improve the argument
